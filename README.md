# JasmineTrellis

A new approach to maintable, dynamic fixture generation for Javascript tests.

## Installation

Add this line to your application's Gemfile:

    group: :development, :test do
      gem 'jasmine_trellis'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jasmine_trellis
